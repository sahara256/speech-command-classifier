# Lightweight Speech Command Classifier for Edge Deployment

## Project Overview

This project implements a lightweight offline speech command classifier designed for automotive and edge AI applications.

The system receives a text command generated by an Automatic Speech Recognition (ASR) engine and predicts the corresponding intent using semantic sentence embeddings and a Logistic Regression classifier.

The primary objective is to build an efficient intent classification system that is suitable for deployment on resource-constrained edge devices.

---

# Project Objectives

- Build an offline speech command classifier
- Perform semantic intent classification
- Detect Out-of-Scope (OOS) commands
- Improve robustness using data augmentation
- Support lightweight edge deployment
- Export the model to ONNX format
- Evaluate inference speed and model size

---

# System Pipeline

```text
Speech
   │
   ▼
Automatic Speech Recognition (ASR)
   │
   ▼
Sentence Embedding (MiniLM)
   │
   ▼
Logistic Regression Classifier
   │
   ▼
Intent Prediction
```

> This project focuses on the **Intent Classification** stage of the complete speech pipeline.
