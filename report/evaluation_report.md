# Evaluation Report

## Lightweight Speech Command Classifier for Edge Deployment

---

# 1. Introduction

The objective of this project was to design a lightweight offline speech command classifier suitable for deployment on edge devices. The classifier predicts user intents from text generated by an Automatic Speech Recognition (ASR) system.

The project emphasizes low memory usage, low inference latency, semantic understanding, and extensibility.

---

# 2. Project Workflow

The implemented pipeline consists of:

1. Dataset preparation
2. Data preprocessing
3. Dataset splitting
4. Data augmentation
5. Sentence embedding generation
6. Logistic Regression training
7. Out-of-Scope detection
8. ONNX export
9. Quantization
10. Interactive inference

---

# 3. Dataset

- OOS samples: 300
- Total samples: 800

---

# 4. Embedding Model

Sentence embedding model:

all-MiniLM-L6-v2

Embedding dimension:

384

---

# 5. Classifier

Algorithm:

Logistic Regression

Advantages:

- Lightweight
- Fast inference
- Easy deployment
- Small memory footprint

---

# 6. OOS Detection

Two methods were implemented.

1. Direct prediction of the OOS class.

2. Confidence threshold of 0.30.

Commands below the confidence threshold are classified as unknown.

---

# 7. Data Augmentation

Noise augmentation was applied to improve robustness against ASR-style variations.

Examples include:

- Word substitutions
- Alternative phrasing
- Minor textual variations

---

# 8. Model Export

The trained classifier was exported to ONNX format and evaluated using ONNX Runtime.

A quantized ONNX model was also generated for deployment experiments.

# 11. Error Analysis

Most prediction errors occurred between semantically similar command pairs such as:

- open_sunroof ↔ close_sunroof
- activate_do_not_disturb ↔ deactivate_do_not_disturb

These errors can be reduced with additional training samples.

# 12. Strengths

- Offline execution
- Lightweight model
- Fast inference

- OOS detection

---

# 13. Limitations

- English-only commands
- Text input only
- Small dataset
- No live ASR integration
- No hardware deployment

---

# 14. Future Work

Future improvements include:

- Whisper ASR integration
- Real-time microphone input
- Mobile deployment
- Multilingual support
- Larger datasets
- Deep learning intent classifiers

---

# 15. Conclusion

A lightweight speech command classifier was successfully developed for edge deployment.

The system achieved **90% test accuracy** while maintaining a small model size and supporting semantic intent classification, Out-of-Scope detection, ONNX export, and interactive inference.

The modular architecture allows new commands to be added with minimal changes, making the system suitable for future automotive and IoT voice assistant applications.
